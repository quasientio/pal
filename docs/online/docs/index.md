## Commands

* `pal run` - Run a new peer.
* `pal print` - Print messages from peers or logs.
* `pal call` - Send messages to peers or logs.
* `pal ls` - List peers and logs in directory.
* `pal rm` - Remove peers or logs from directory.
* `pal help` - Display help information about the specified command.
